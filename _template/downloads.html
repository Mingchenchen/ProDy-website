Installation

0 Using a Package Manager

You can install ProDy and Evol using pip, pip install ProDy==1.7

Windows users need to use pre-compiled versions from section 2.

1 Requirements
You need to install the following software before installing ProDy:
Python
We recommend that you use 2.7, 3.3 or later. For Windows, choose a 32-bit or 64-bit Python installer.
NumPy
We recommend that you use the latest version of NumPy, but v1.7 and later are supported. For Windows, make sure that you choose compatible versions of NumPy and ProDy.

3 Installation
  • Linux & Mac OS
  • Windows

Extract tarball contents and run setup.py as follows:

$ tar -xzf ProDy-{{ version }}.tar.gz
$ cd ProDy-{{ version }}
$ python setup.py build
$ sudo python setup.py install

4 Recommendations
We recommend that you also install Matplotlib for plotting, and IPython for interactive usage. On Windows, you need to install pyreadline for colorful IPython sessions.
