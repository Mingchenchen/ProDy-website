coMD for Collective MD Analysis

coMD is a VMD plugin designed for setup collective molecular dynamics simulations with elastic network models for conformational sampling. coMD can incorporate a diverse set of molecules from CHARMM protein and lipid force field (C22 and C27 lipids) into simulations.

Features

coMD utilizes various softwares in following order:

  1. Using VMDâ€™s psfgen and solvate packages, solvate and ionize protein structures.
  2. Run a short minimization on both structures (initial endpoints) using NAMD.
  3. Using ProDy ANM class to sample a series of collective motions to generate a new target conformation at both ends based on Monte-Carlo Metropolis algorithm with ANM modes.
  4. Run a TMD simulation to target conformation using NAMD.
  5. Run a short minimization on final structures using NAMD.
  6. Check whether the RMSD between the two instantaneous endpoints is less than 1.5A or whether the change in the RMSD is less than 0.15 A using VMD. If true, write trajectory files, else, back to step 3.

Download

coMD plugin files can be obtained from Getting coMD.
