MechStiff for Mechanical Stiffness Calculations

MechStiff is a method that calculate the mechanical resistance of protein to external force applied at specific positions of 3D structure.

Features

MechStiff use as an input ANM instance and protein coordinates. With this tool multiple features may be exhibited, a.o.: Mechanical Stiffness Map, Mean value of stiffness: as plot with secondary structure or as a color coding 3D structure (TCL script directly uploaded to VMD). All plots and vizualization are automatically provided. It can be used for:

  1. Localization of the weakest/strongest elements of the structure architecture provided together with 3D vizualization and statistics analysis.
  2. Determing the value of effective spring constant for selected pair of residues - useful for Single Molecule Force Spectroscopy (SMFS, AFM) and Steered Molecular Dynamics simulations.
  3. Studing distributions of the deformation in the distance contributed by each mode in the presence of extensional forces applied to the pair of residues.
Reference
Eyal E, Bahar I Toward a Molecular Understanding of the Anisotropic Response of Proteins to External Forces: Insights from Elastic Network Models Biophys J 2008 94(9):3424-3435.
